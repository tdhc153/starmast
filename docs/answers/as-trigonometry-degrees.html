Answers: Trigonometry (degrees)

Author

Dzhemma Ruseva, Ellie Gurini, Ciara Cormican

Summary
Answers to the questions on trigonometry, using degrees to measure angles.

These are the answers to Questions: Trigonometry (degrees).

Please attempt the questions before reading these answers!

Q1

You are given the triangle below.

Here,

  • \(\cos(a) = \dfrac{4}{5}\)

  • \(\sin(a) = \dfrac{3}{5}\)

  • \(\tan(a) = \dfrac{3}{4}\)

  • \(\cos(b) = \dfrac{3}{5}\)

  • \(\sin(b) = \dfrac{4}{5}\)

  • \(\tan(b) = \dfrac{4}{3}\)

Q3

3.1. \(\cos(30) = \dfrac{\sqrt{3}}{2}\)

3.2. \(\tan(30) = \dfrac{1}{\sqrt{3}} = \dfrac{\sqrt{3}}{3}\)

3.3. \(\csc(45) = 1\)

3.4. \(\cot(30) - \sin(60) = \sqrt{3} - \dfrac{\sqrt{3}}{2} = \dfrac{\sqrt{3}}{2}\)

3.5. \(\sin(90) + \cos(180) = 1 + (-1) = 0\)

3.6. \(\tan(30) - \cot(30) = \frac{1}{\sqrt{3}} - \sqrt{3}\)

3.7. \(\cos(0)\sin(90) = 1\cdot 1 = 1\)

3.8. \(\cos(30)\sec(30) - \sin(45)\csc(45) = 1 - 1 = 0\)

3.9. \(\cot(90) = 0\)
